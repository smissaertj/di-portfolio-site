Vim Quickstart

Nano used to be my favorite command line text editor until I (re)discovered Vim (Vi improved). I never was a big fan of Vim earlier on because of the command and input mode that leads to confusion. One does not simply quit Vim. Well, at least not until you put your back into it.

I decided to take some time to learn Vim because it offers some enhancements over regular Vi (syntax highlighting for one) and Vi is really the only text editor that is always available on any Linux distribution.
Everything I mention here on Vim will be applicable for Vi as well.

After an hour or two getting my feet wet it all started to make sense and the commands started sticking around. Vim has now become my goto text editor on the command line, I even created an alias for Nano to Vim in my .bashrc file just in case.

I started getting comfortable with Vim by using vimtutor. This command opens a Vim tutorial that let’s you work through some exercises.
You can install vimtutor by doing dnf install vim-enhanced.
